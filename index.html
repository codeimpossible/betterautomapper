Better Auto Mapper

A library to help you map one object, to another with as little code as possible.

Examples

using System;
using BetterAutoMapper;

namespace MyProgram
{
    public class Animal
    {
        public string Name { get; set; }
        public int Limbs { get; set; }
    }

    public class Cat
    {
        public string Name { get; set; }
        public string Limbs { get; set; }
        public bool HasTail { get; set; }
    }

    class Program
    {
        static void Main(string[] args)
        {
            var animal = new Animal { Name = "Fred", Limbs = 4 };
            var cat = BAM.BOOM.Map<Animal, Cat>(animal);

            Console.WriteLine(cat.Name);
            Console.ReadLine();
        }
    }
}

Features

  • Case Insensitive Mapping. someProperty will map to SomeProperty or even SomEpRopertY
  • Strict maps. Want to know if you're losing data when mapping? We've got that.
  • Property Transposing. Want FullName to be FirstName + " " + LastName? That's easy. Want to transform data using some of your own code? Yep, totally do-able.
